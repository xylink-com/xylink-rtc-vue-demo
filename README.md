# XYLINK WEBRTC SDK

小鱼易连 webrtc sdk demo 项目。

此项目使用到的技术栈：vue

## 准备工作

1. 注意在`src/utils/config.js`中配置第三方登录所需的

- extId：企业 id
- clientId： Client id

## 运行

### 安装依赖

```bash
$ yarn
or
$ npm install
```

### 运行

```bash
$ yarn start
or
$ npm run start
```

所用到的工具自行学习了解：

1. [yarn](https://yarn.bootcss.com/)
2. [node](https://nodejs.org/zh-cn/)
3. [vue](https://vuejs.org/)
